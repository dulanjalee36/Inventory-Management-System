# Inventory Management System

## 📌 Description

The **Inventory Management System** project is a comprehensive desktop application designed to help businesses efficiently manage their inventory, suppliers, employees, and sales.

Built using **Python** with a **Tkinter graphical user interface**, this system provides a user-friendly platform for organizing and tracking stock items while automating routine tasks.

### 🔑 Key Highlights

- **User Authentication** – Secure login for authorized access.
- **Product Management** – Add, edit, delete, and search products with detailed information.
- **Category & Supplier Management** – Manage product categories and supplier details.
- **Employee Management** – Maintain employee records.
- **Sales Management** – Record sales transactions and generate reports.
- **Dashboard** – Interactive dashboard displaying real-time statistics.

This system is ideal for small to medium-sized businesses looking to reduce manual record-keeping, improve stock monitoring, and make data-driven inventory decisions.

---

## 🚀 Features

### 1️⃣ User Login & Authentication
- Secure login system.
- Protects sensitive inventory and sales data.

### 2️⃣ Product Management
- Add new products (name, category, supplier, price, stock quantity).
- Edit and update product details.
- Delete products from inventory.
- Search and filter products quickly.

### 3️⃣ Category Management
- Create, edit, and delete categories.
- Organize products effectively.

### 4️⃣ Supplier Management
- Maintain supplier details (name, contact, address).
- Add, update, or remove suppliers.

### 5️⃣ Employee Management
- Store and manage employee information.
- Track employee roles in inventory and sales.

### 6️⃣ Sales Management
- Record sales transactions.
- Automatically update stock after each sale.
- Generate sales reports.

### 7️⃣ Dashboard & Analytics
- Visual dashboard showing:
  - Total products
  - Sales statistics
  - Employee count
- Quick business performance overview.

### 8️⃣ User-Friendly Interface
- Intuitive Tkinter GUI.
- Easy navigation with buttons and forms.

### 9️⃣ Data Persistence
- All data stored securely in a **MySQL database**.

### 🔟 Error Handling & Alerts
- Pop-up messages for:
  - Successful operations
  - Warnings
  - Errors
- Prevents incorrect entries and accidental data loss.
